The Create User Accounts interface allows administrators to add new team members to the operational system. Set up user profiles, assign roles, and configure team memberships through a streamlined creation process.

## Who can create user accounts

<Badge text="Admin" variant="danger" /> Full access to create any type of user account

<Badge text="Ops Support" variant="caution" /> Can create operational user accounts with approval

<Badge text="All Others" variant="note" /> Cannot create user accounts

## What is Create User Accounts?

The user creation system provides:
- **Profile Setup** - Basic user information and contact details
- **Role Assignment** - System role selection based on job function
- **Team Assignment** - Organization into functional teams
- **Account Activation** - Immediate system access upon creation

## Best Practices

:::tip[User Creation Best Practices]
- **Verify Information:** Double-check all personal details before creating account
- **Appropriate Role Assignment:** Match role to actual job responsibilities
- **Team Alignment:** Ensure team assignment matches organizational structure
- **Immediate Communication:** Contact new user with login details promptly
- **Follow-up:** Check that new user can access system and required features
:::

:::note[Integration Points]
Create User Accounts integrates with:
- **Account Management:** New users appear in the main user table
- **Configuration Centre:** Role permissions are applied from role definitions
- **Team Management:** Team assignments affect user visibility and access
- **All System Features:** Role-based access controls user's system experience
:::

## Security Considerations

:::warning[Security Best Practices]
- **Role Verification:** Ensure new users receive only necessary permissions
- **Email Validation:** Verify email addresses belong to legitimate users
- **Access Reviews:** Regularly review new user access and permissions
- **Documentation:** Keep records of who created accounts and when
:::

## Getting Started

To create a new user account:

1. **Access the form** via Admin menu > Create User Accounts
2. **Enter personal details** - First Name, Last Name, and Email
3. **Select appropriate role** from the Role dropdown
4. **Choose team assignment** from the Team dropdown
5. **Click "Create Account"** to complete the process
6. **Verify account creation** in Account Management
7. **Provide login details** to the new user
