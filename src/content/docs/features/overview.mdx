## All Available Features

### Tracking & Monitoring
- **Dashboard** - Central operational overview
- **Operational Tracker** - Monitor operational days and performance
- **CIPHR Integration** - Process holiday data and resolve conflicts
- **Lieu & Holidays** - Track holiday requests and lieu time

### Scheduling
- **Rota Management** - Complete workforce planning system
  - Event Mapping - Coordinate events with resources
  - Contract Allocation - Assign contracts to teams
  - Succession Planning - Plan for role transitions
- **Schedules** - Manage individual and team schedules

### Management
- **Request Management** - Submit and process operational requests
- **EOS Actions** - Manage operational tasks and workflows
- **Allocated Contracts** - View and manage contract assignments
- **Assigned Events** - Coordinate event assignments
- **Create User Accounts** - Add new system users
- **User & Team Management** - Organize teams and permissions
- **Configuration Centre** - System settings and configuration

## Role-Based Feature Access

| Feature Category | Admin | Ops Support | Ops Manager |
|------------------|-------|-------------|-------------|
| **Tracking** | Full Access | Full Access | Limited |
| **Scheduling** | Full Access | Full Access | Team Only |
| **Management** | Full Access | Most Features | Limited |
