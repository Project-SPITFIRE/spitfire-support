## Best Practices

:::tip[CIPHR Integration Tips]
- **File Format:** Ensure CIPHR exports are in .xlsx format
- **Regular Uploads:** Process CIPHR data regularly to minimize conflicts
- **Conflict Resolution:** Address conflicts promptly to maintain schedule accuracy
- **Manager Communication:** Coordinate with line managers on scheduling conflicts
:::
