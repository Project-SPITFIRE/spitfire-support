Manage operational contracts, monitor event allocations, and track operational days for your team.

## Accessing Contract Allocation

1. Navigate to **Scheduling** → **Rota Management**
2. Select **Contract Allocation** from the submenu

## Page Actions

**Search Contracts**: Find specific contracts quickly
**+ Create New Contract**: Add new contracts  
**Card View / Table View**: Toggle display modes

## View Modes

## Understanding Contract Information

Each contract displays:
- **Contract Name**: Descriptive identifier
- **Events count**: Number of events assigned
- **Operational Days**: Current usage vs. allocated (e.g., "3/100")

## Contract Management

<Accordion>
  <AccordionItem title="Viewing Contract Events">
    <p>
      Locate contract card → Click blue <strong>Events</strong> button → Review assigned events → Check dates, managers, status → Make adjustments
    </p>
  </AccordionItem>
  <AccordionItem title="Editing Contracts">
    <p>
      Find contract → Click <strong>Edit Contract</strong> → Update name, allocations, terms → Adjust operational day limits → Save changes
    </p>
  </AccordionItem>
  <AccordionItem title="Creating New Contracts">
    <p>
      Click <strong>+ Create New Contract</strong> → Fill details (name, days, period, client) → Set special requirements → Save contract
    </p>
  </AccordionItem>
</Accordion>

## Best Practices

<Accordion>
  <AccordionItem title="Regular Monitoring">
    <p>
      Check operational day usage weekly → Watch for approaching limits → Plan assignments based on available days → Communicate capacity issues
    </p>
  </AccordionItem>
  <AccordionItem title="Capacity Planning">
    <p>
      Review upcoming events vs available days → Identify potential issues early → Negotiate extensions when needed → Balance across multiple contracts
    </p>
  </AccordionItem>
  <AccordionItem title="Documentation Standards">
    <p>
      Keep clear, descriptive names → Include client/venue identifiers → Document special requirements → Note assignment restrictions
    </p>
  </AccordionItem>
</Accordion>

## Common Scenarios

<Accordion>
  <AccordionItem title="Approaching Day Limits">

:::note[Contract Capacity]
Monitor operational day usage to avoid exceeding allocations. Reaching limits may require contract amendments or affect ability to fulfill additional requests.
:::
